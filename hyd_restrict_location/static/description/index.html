Restrict operation/move on location

This module give the manager of the warehouse the possibility to restrict move to a location. By example, if your are the manager you would not want simple user of warehouse module to make move/opration from and to inventory loss or scrapped. This module allow you to put your user account as the one who can do inventory and move stock to the scrapped location.
